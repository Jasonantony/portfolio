On Going Projects

AUTO VENUMA

"Auto Venuma" is an ongoing project aimed at streamlining local transportation through a dedicated auto-rickshaw booking app. Designed for urban and semi-urban areas, it allows users to locate nearby autos, estimate fares, and book rides instantly. The app focuses on simplicity, real-time tracking, and local language support to make auto travel more accessible and organized.

AT TRAVELS

"AT Travels" is an in-progress bus ticket booking platform tailored for regional and long-distance travel. It aims to allow users to search routes, view available buses, choose seats, and make secure bookings. The application is being built with a focus on clear UI, booking history, cancellation options, and integration with payment gateways.

Cab Booking Application

This project is a prototype of a cab booking system that allows users to schedule rides, track cabs live, and receive driver updates. The plan includes user authentication, trip history, fare calculators, and map integration. The backend is powered by Python, while the frontend is being developed with HTML, CSS, and JavaScript.
